Introduciton to CSS

  • Cascading Style Sheets (CSS)
    • Rules that specifiy how the content of an element should appear
  • "Invisible Box around all HTML Elements:

CSS Associations

  • Selector: which eleent the rule applies to
  • Declarations: how the element should be styled

CSS Properties

  • Properties: aspect of the element you want to change
  • Value: specify the setting you want to use for the property

External CSS

  • ≤link≥ tag
    • href specifies the path to the CSS file
    • types specifies the file type
    • rel specifies the relationship between the HTML and the CSS file

Internal CSS

  • ≤style≥ start tag
  • ≤/style≥ end tag
  • Used inline or within the HTML code

CSS Selectors

  • Universal
    • Applies to all elements in the document
  • Type
    • Matches element names
  • Class
    • Matches an element with class attributes
  • ID
    • Matches an element with ID attributes
  • Child
    • Matches an element that is a direct child of another
  • Descendant
    • Matches an element that is a descendant of another
  • Adjacent Sibling
    • Matches an element that is the next sibling
  • General Sibling
    • Matches an element that is a sibling of another

How CSS Rules Cascade

  • Important when two or more rules are present, which will take prcedence
  • Last Rule
    • Only the later of the two or more rules will apply
  • Specificity
    • Only the more specific rule will apply
  • Important
    • If "important" is added it will take precedence

Link Text Attributes

  • Link
    • Style links not yet visited
  • Visited
    • Style links already visited
  • Hover
    • Style links when user hovers over the link
  • Active
    • When a user clicks on a link
  • Focus
    • When an element has focus
