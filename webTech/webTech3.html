Lists

  • Ordered Lists

    • List item position important

    • Numbered or alphabetized

    • Steps to follow

  • Unordered Lists
    • List item position unimportant
    • Bullet points
  • Definition Lists
    • Terms with definitions

Ordered Lists

  • ol start tag
  • /ol end tag
  • List item
    • All browsers indent list items by default
    • li start tag
    • /li end tag

Unordered Lists

  • ul start tag
  • /ul end tag
  • List item
    • All browsers indent list items by default
    • li start tag
    • /li end tag

Definition Lists

  • dl start tag
  • /dl end tag
  • Term being defined
    • dt start tag
    • /dt end tag
  • Definition
    • dd start tag
    • /dd end tag

Nested Lists

  • Used when a list within a list is required
  • By default, browsers will
    • Indent the second list
    • Use a different bullet style

Lists Attributes

  • Ordered List: type
    • type="1"
      • Numbered list (default)
    • type="A"
      • Uppercase letters ascending
    • type="a"
      • Lowercase letters ascending
    • type="I"
      • Uppercase roman numerals ascending
    • type="i"
      • Lowercase roman numberals ascending

Lists Attributes

  • Ordered Lists: start
    • Change the default starting index
      • start="10"
    • Only works with default (type="1") type lists

Links

  • Links make the internet work
    • Create links between webpages/websites
    • Linking to other sites and resources
  • Responsible for "surfing" the web
  • Clear links enable a good end user experience and effective site navigation

Link Element

  • ≤a≥ start tag
  • ≤/a≥ end tag
  • Example:
    • ≤a href="URL"≥me.com≤/a≥

Relative Links

  • Linking to other pages within your site
  • Do not require full paths, but will accept them
  • Example
    • ≤a href="home.html"≥Home≤/a≥
    • ≤a href="contact.html"≥Contact≤/a≥
    • ≤a href="about.html"≥About≤/a≥
  • All pages must exist in same folder for relative links to work

Absolute Links

  • Linking to other pages or sites outside of your website
  • Example
    • ≤a href=“http://www.utsa.edu”≥UTSA≤/a≥
    • ≤a href=https://www.facebook.com”≥Facebook≤/a≥

Email Links

  • Start the client's default email program
  • Example
    • ≤a href=mailto:juan.valadez@utsa.edu”≥Email Prof. Valadez≤/a≥

Link Attributes

  • Open a link in a new browser window
    • target="_blank"
  • Anchors
    • ≤a href="#top"≥Top≤/a≥
    • ≤h1 id="Top"≥Home Page≤/h1≥

Images

  • "A picture is worth 1000 words"
  • Objectives
    1. Use HTML to insert images to your web page
    2. Select appropriate image format
    3. Size images accordingly
    4. Optimize images for faster page loads

Image Considerations

  • Copyright laws
    • Cannot assume "Fair use"
  • Stock images
    • Subscription based
  • Images should
    1. Be relevant
    2. Convey information and mood
    3. Be recognizeable
    4. Fit theme

Local Images

  • "images" folder
    • Typical of all websites
    • May contain sub-folders

Image Tag Structure

  • Starting tag: ≤img≥
  • Closing tag: ≤/≥
  • Source: src
    • Location of file: local or remote
  • Description: alt
    • Loads a description in case image is not loaded
  • Title: title
    • toot-tip text
  • Example:
    • ≤img src="images/banner.jpg" alt="Banner Image" title="Main Page Banner"≥

Image Structure: Height & Width

  • Specifiy the image to a particular size
  • Height
    • height in pixels
  • Width
    • width in pixels
  • CSS is used more to control height and width

Image Placement

  • Before a paragraph
  • Inside the start of a paragraph
  • In the middle of a paragraph
  • Alignment
    • Horizontal: left, right
    • Vertical: top, middle, bottom
  • CSS typically used for image placement

Image Creation

  • Save images in right format
  • Save images in right size
  • Measure iages in pixels
  • Image formats
    1. JPEG
    2. GIF
    3. PNG
    4. Vector Images (SVG)

Image Dimensions

  • Reducing image size
    • Help the web page load faster
  • Increase image size
    • May slow web load
    • May make image blurry/blocky
  • Cropping images
    • Adjusting image to reduce unnecessary information

Image Measuremenets

  • Pixels
    • The unit of measurement that makes up images: tiny "squares"
  • Resolution
    • The number of pixels an image is made of
    • 300 DPI (dots per inch) is typical image resolution
  • Web: dimension in pixels
    • PPI (pixels per inch)
    • 72 PPI is standard web dimensions for images

HTML5 Images

  • Figure
    • Starting tag: ≤figure≥
    • Ending tag: ≤/figure≥
  • Figure Caption
    • Starting tag: ≤figcaption≥
    • End tag: ≤/figcaption≥
