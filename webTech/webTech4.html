Basic Table Structure

  • Creating a table
    • Start tag: ≤table≥
    • End tag: ≤/table≥
  • Table Heading
    • Start tag: ≤th≥
    • End tag: ≤/th≥
    • Attribute: scope
      • Row
      • Column
  • Used as "title" information, usually bold
  • CSS may alter the look and feel of a table heading

Table Rows

  • Rows
    • Start tag: ≤tr≥
    • End tag: ≤/tr≥
  • Row Attributes
    • Spanning Rows:
      • rowspan=""
    • Background colors
      • bgcolor="#FF0000

Table Columns

  • Column
    • Start tag: ≤td≥
    • End tag: ≤/td≥
  • Column Attributes
    • Spanning Columns:
      • colspan=""
    • Background colors
      • bgcolor="#FF0000"

Long Tables

  • Table Heading
    • Start Tag: ≤thead≥
    • End Tag: ≤/thead≥
  • Table Body
    • Start Tag: ≤tbody≥
    • End Tag: ≤/tbody≥
  • Table Footer
    • Start Tag: ≤tfoot≥
    • End Tag: ≤/tfoot≥

Table Attributes: part 1

  • Width
    • Defined in percentage of page or in pixels
    • width="400px" or width="50%"
  • Cell Padding
    • Determine the space within the cell and the data in pixels
    • cellpadding="10"
  • Cell Spacing
    • Determine the space seperating each cell from other cells in pixels
    • cellspacing="10"

Table Attributes: part 2

  • Border
    • Indicate the width of the table border in pixels
    • border="2"
  • Background Color
    • Determine the background color of entire table or columns or cells
    • bgcolor="#FF0000"
